## 📋 Project Overview

This project replaces manual Excel-based employee tracking at CSIR with
a professional, cloud-connected web portal. It provides real-time
visibility into employee records, department headcount, and
organizational statistics.

**Intern:** Mulagalapatilokesh
**Organization:** CSIR
**Duration:** 2026
**Stack:** MongoDB, Express.js, React.js, Node.js (MERN)

---

## ✨ Features

- 📊 Real-time dashboard with KPI cards and bar charts
- 👥 Complete employee CRUD (Create, Read, Update, Delete)
- 🔍 Live search and filter by status
- ✅ Form validation with error messages
- 🗄️ MongoDB Atlas cloud database
- 🧪 API tested with Postman — 8/8 tests passing
- ⚡ Load tested with Apache JMeter — 50 concurrent users
- 📱 Professional dark sidebar UI
- ⚙️ Settings saved permanently to database
- 📁 Full GitHub version control history
